# 🌦️ Weathix: Architectural Weather Intelligence

Weathix is a premium, privacy-focused weather dashboard designed with a high-contrast **Architectural Ledger** aesthetic. It provides precise meteorological data, interactive maps, and an AI-powered weather assistant to help you plan your day with confidence.

---

## 🚀 Quick Start

### 1. Prerequisites
- **Node.js** (v16 or higher)
- **npm** or **yarn**

### 2. Installation
Clone the repository and install the dependencies:
```bash
git clone https://github.com/CLANKAK-DEV/web2.git
cd web2
npm install
```

### 3. Configuration
Create a `.env` file in the root directory and add your API keys:
```ini
REACT_APP_OPENWEATHER_API_KEY=your_key_here
REACT_APP_VENICE_API_KEY=your_key_here
```

### 4. Launch
Start the development server:
```bash
npm start
```
The app will be available at `http://localhost:3000`.

## 🛠️ How to Use Weathix

### 🔍 Searching for a City
- Click the **Search** input in the top bar.
- Type any city name (e.g., "Paris", "Casablanca").
- Select your city from the results to load its full meteorological ledger.

### 📍 Using "Locate Me"
- Click the **Locate Me** button in the sidebar.
- If it's your first time, allow the browser to access your location.
- **Tip**: If you accidentally denied permission, a helpful modal will appear with instructions on how to re-enable it in your browser settings.

### 🗺️ Exploring the Radar
- Click the **Map** tab in the sidebar.
- Toggle between **Precipitation**, **Wind**, and **Clouds** using the buttons at the top of the map.
- Pan and zoom to see atmospheric movements across the globe.

### 🤖 Chatting with the AI Assistant
- Click the **AI Assistant** bubble in the bottom right corner.
- Ask questions like: *"What should I wear today?"* or *"Will it rain during my outdoor wedding on Saturday?"*
- The AI analyzes the current dashboard data to give you specific, expert advice.

### 📋 Managing Saved Locations
- Click the **Saved Locations** tab.
- Here you can see a "Register" of all your favorite cities.
- Use the **Sort** and **Filter** options to organize your list by name or country.
- Click **Add New City** to open the global search modal.
